　　·互联星空概述
　　互联星空 是中国电信推出的一项互联网应用服务。用户通过互联星空可以享受到影视、游戏、教育等丰富多彩的互联网内容服务。同时，互联星空还是一种方便快捷的支付手段。

　　·互联星空一点通
　　互联星空的服务特色主要是“一点认证，全网通行”。江苏电信宽带用户使用自己的宽带账号访问互联星空服务网站或互联星空加盟网站时，无须再输入账号和密码，只需点击“互联星空一点通”按钮，即可安全登录。
　　其标识为：
　　
　　互联星空一点通的最大特点是安全快捷、方便。具体表现为；无需牢记长长的帐号和密码；鼠标一点即通，无需再次键盘输入帐号、密码、验证码，无需等待。

　　“我的星空”服务
　　用户登录后，进入“我的星空”可以查看和修改个人资料，包括个人信息、密码的修改及安全设置等；可以查看用户的账户信息，了解用户消费的明细账或进行账号充值；可以查看已订购的服务、取消续订、关闭托收等，或对不满意的服务进行在线投诉。

　　特色频道
　　江苏互联星空拥有丰富多彩的特色频道，如观看影视剧的星空院线和星空剧场、保障电脑安全的星空杀毒、浏览时尚资讯的星空杂志等。用户可以定制自己喜欢的服务，详情请咨询所在地的10000号。

　　天翼视讯
　　江苏互联星空特别推出天翼视讯“星空院线”、“星空剧场”频道，其中影视资源非常丰富，且频繁更新国内外近期上映或经典好看的大片、热剧。两个影视频道面向e家尊享用户免费开放（普通e家用户在前台办理e家娱乐服务包，也可享受免费影视服务）；对普通宽带用户收取一定费用，具体资费为：星空院线包月21元/月，点播普通影片5元/部、分集影片3元/部；星空剧场包月20元/月，点播电视剧1元/集。按次点播节目有效观看时间为48小时。

注意：

支付密码申请
为加强互联星空消费安全性，特别通过与电话捆绑的方式，对互联星空大额收费服务进行二次密码认证。用户在互联星空消费的过程中，需输入一个支付密码进行确认。该密码需注意保管，请勿将其设置为与宽带上网密码相同。

安全设置
江苏电信宽带用户可以设置下次登录互联星空是否使用“一点通”功能。选择“允许”并保存修改，则用户下次登录时可使用“互联星空一点通”方式、一点即上；选择“不允许”并保存修改，则用户登录时不能使用“互联星空一点通”方式，每次登录互联星空都需输入宽带接入账号、互联星空服务密码和附加码。
注意：设置前请用户确认知道自己的互联星空服务密码，以免出现无法登录的情况。如果用户没有进行安全设置，系统将默认为“允许”，即用户登录可以使用互联星空一点通方式。
